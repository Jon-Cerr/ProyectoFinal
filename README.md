# Proyecto Final - Clon de Mortal Kombat en C
Este repositorio alberga el codigo fuente de un clon basico de Mortal Kombat para la entrega de un <br/>Proyecto Final de la materia **Programacion Estructurada**
## Caracteristicas principales - herramientas de C utilizadas en este proyecto
* Memoria dinamica y punteros
* Matrices y arreglos
* Structs y enums
* Sentencias de control
* Archivos
* Comuniacion serial
